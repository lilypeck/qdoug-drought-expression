# qdoug-drought-expression
Scripts for analysis of transcriptomics in _Quercus douglasii_ drought and well-watered treatments. 

A repository for all custom analysis and plotting scripts for the manuscript 'Drought-associated genes exhibit high constitutive expression in _Quercus douglasii_, a drought tolerant California oak'
